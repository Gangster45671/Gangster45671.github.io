JS, Flash and Java: Interactions

After HTML and CSS were added came Java Script, Flash and Java applets. This added interaction, and client side scripting to websites. This meant that websites could now run scripts/apps on your computer. This was the 'action' that was previously missing. Unfortunately you cannot edit Java Script live, so I have pre-done it for you. By inserting <button onclick="button()" id="button">Click Me!</button> you will render a button. Via the 'onclick' event propierty you can trigger a Java Script function. Which can then execute and do more complex tasks, such as showing and alert or changing the content of the button.
This is acctually what enables the live editor you see aside to work.

This begun a brand new era, now you could make games, calculators, and many more interactive sites. This added a whole new dimension of what website could do. Now not only could websites display information, but it could also process it live.
This had a huge impact on the internet, this is the basic foundation for acctually usefull websites to come to life. This was the point where people started to think as websites not only as information display tools, but also as application, game, schedule manager, search engine.. anything.
