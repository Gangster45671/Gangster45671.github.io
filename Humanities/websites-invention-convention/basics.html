HTML and CSS: The basics

Web pages and websites are composed of many different coding languages, but the main ones are HTML and CSS. HTML stands for Hyper Text Markup Language, and CSS is Cascading Style Sheets. HTML is an element/tag based language, this means that there are different elements. Copy and paste any code to the editor on the right to try it out. For example a paragraph is a <p> tag, and continues until a </p>. Every HTML element is a tag and must be enclosed. The content goes in between the tags. Like so <p> sample paragraph </p>. Tags also have proierties, like links: <a href="link-goes-here.com">This is a link</a>. Furthermore not all tags render elements, some are just for style for example <center>. Try expirimenting with some other tags: 

<h1> header </h1>
<br /> <!-- that was a line break, this is a comment -->
<p> paragraph </p>
<b> bold text </b>
<i> italic text </i>
<center> <p> This is centered </p> </center>

As the name implies, CSS styles webpages. It changes how elements are rendered onscreen, a key difference is that CSS does not render and display on the page only the HTML does. Here is an example; if you want to make the colour of all headers red 

h1 {
  color: red;
}

You can also add CSS inline, as a tag. But this is not preferable as it makes the code harder to read. Here is a colorful poem: 

<p>
    Roses are <span style="color:red">red</span>.
    <br />
    Violets are <span style="color:blue">blue</span>.
    <br />
    HTML is <span style="color:green">cool</span>,
    <br />
    and CSS is <span style="color:green">too</span>!
</p>

This was all the first website had to use. HTML and CSS were the only features available. Eventough HTML and CSS are extremely dynamic and allow you to display litterally anything you want, they are not very functional. It is just a still render.

This was already good enough to make information shareable online. It might not seem like much but everything starts somewhere...
